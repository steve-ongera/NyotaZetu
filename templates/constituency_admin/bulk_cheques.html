Bulk Cheque Instructions

What are Bulk Cheques?

Bulk cheques consolidate payments for multiple students attending the same institution into a single cheque. This reduces administrative overhead and processing costs.

Creating a Bulk Cheque

  1. Click the "Create Bulk Cheque" button above
  2. Select the institution for which you want to create a bulk cheque
  3. Select the students whose allocations should be included
  4. Enter cheque details (number, holder information)
  5. Review and confirm the total amount
  6. Submit to create the bulk cheque

Cheque Collection

  • The designated cheque holder must present valid identification
  • Verify the cheque holder's identity before releasing the cheque
  • Record the collector's ID number when marking as collected
  • All collection activities are logged for audit purposes

Important Notes

  • Only approved and allocated applications can be included in bulk cheques
  • Each student will receive a notification when included in a bulk cheque
  • Bulk cheques must be collected within 90 days of creation
  • Ensure all bank details are correct before creating the cheque
